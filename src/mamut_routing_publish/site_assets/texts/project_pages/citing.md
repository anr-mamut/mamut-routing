# Citing MAMUT-routing

If you use `MAMUT-routing`, its benchmark artifacts, or accompanying tools such as the [`mamut-routing-lib` Python library and CLI](https://github.com/ANR-MAMUT/MAMUT-routing-lib), we would appreciate citations to the corresponding software and publication records.

This page is a work in progress while the first public release is being prepared.

## Current Citation Guidance

Until DOI, Software Heritage, HAL, and publication records are available, please cite the public repository once it is released and mention the version or Git commit used in your experiments.

## Future Software Release

The intended release workflow is:

- publish the GitHub repository;
- create a `v0.1.0` GitHub release;
- archive the release in Software Heritage;
- create a HAL software entry;
- update this page with stable links and a BibTeX entry.
