# Funding

MAMUT-routing is developed in the context of the ANR-MAMUT project, ANR-22-CE22-0016.

The ANR-MAMUT project is titled "Machine learning et matheuristiques pour le transport urbain - MAMUT" and is funded under the CE22 call for transports, mobility, and construction in urban and peri-urban territories.

## Project Record

- Project code: `ANR-22-CE22-0016`
- Official record: [ANR project page](https://anr.fr/Projet-ANR-22-CE22-0016)
- Coordinator: Marc Sevaux, Universite Bretagne Sud
- Partners: CITI EA3720, Mapotempo, LAB-STICC IMT Atlantique, LAB-STICC Universite Bretagne Sud
- ANR support: 497,722 euros
- Scientific period: 2023 - 2026

## Relation To MAMUT-routing

MAMUT-routing contributes to the ANR-MAMUT goal of making routing research more reusable by exposing benchmark artifacts, provenance, objective contracts, solution files, and visualization tools through a maintainable open-science software infrastructure.
