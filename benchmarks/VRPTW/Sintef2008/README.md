# Sintef2008 benchmark family

This directory contains the MAMUT-routing curation of the classical
Solomon/Gehring-Homberger VRPTW instances under the SINTEF 2008 hierarchical
objective convention.

  relevant rights.

  third-party historical benchmark material. This directory-level notice does
  not relicense those third-party sources.

when redistributing this benchmark family.
